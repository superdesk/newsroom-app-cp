Utilisez notre Recherche Avancée pour obtenir des résultats de recherche plus ciblés sans avoir besoin d’apprendre à rédiger une requête de recherche avec expressions conditionnelles

Toutes ces conditions:

  • Argument classique «AND»
  • Chaque mot inclus doit apparaître dans les résultats
  • C’est une recherche plus restrictive

Par exemple : Pommes Poires = Pommes AND Poires

N’importe laquelle des conditions:

  • Argument classique «OR»
  • Chaque mot inclus ne va pas apparaître dans les résultats
  • C’est le genre de recherche le plus large

Par exemple : Pommes Poires = Pommes OR Poires

Aucune de ces conditions:

  • Argument classique «NOT»
  • Chaque mot inclus doit être absent dans les résultats
  • C’est la recherche la plus restrictive
  • Utilisez ce regroupement avec d’autres mots clés et autres champs de recherche pour affiner les résultats de recherche

Par exemple : Pommes Poires = NOT Pommes NOT Poire

Les caractères génériques et les guillemets peuvent être utilisés dans les trois champs de recherche avancée

  • Vous pouvez utiliser le caractère générique ? pour remplacer un caractère seul ou * pour remplacer aucun ou plusieurs caractères, ainsi que les guillemets «» pour trouver une phrase exacte
